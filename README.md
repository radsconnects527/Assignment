# Summary

Verizon Wireless Assignment.

# Technologies used

HTML, SASS, AngularJS, Bootstrap, Yeoman, Gulp, Bower, Karma, Jasmine, Protractor, node.js

# Project Prerequisites

* node.js
* git

# How to run?
```sh
$ npm install 
$ npm install -g gulp
$ bower install
$ gulp serve
```
# How to unit test?
```sh
$ npm install -g karma
$ karma start
```
# How to integration test?
```sh
$ npm install -g protractor
$ protractor test/conf.js
```
